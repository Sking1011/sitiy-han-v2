# Функциональные спецификации

## 1. Управление складом
- Просмотр текущих остатков (сырье, упаковка, готовая продукция).
- Категории и подкатегории для удобного поиска.
- Уведомления о критически низком уровне запасов.

## 2. Финансовый модуль
- Валюта: Казахстанский тенге (KZT).
- Учет "грязной" прибыли (выручка).
- Учет затрат на закуп сырья и доп. расходы.
- Расчет "чистой" прибыли.
- Графики и диаграммы (динамика продаж, структура расходов).

## 3. Калькуляция (Рецептуры)
- Конструктор техкарт: указание веса/количества каждого ингредиента на единицу продукции.
- Автоматический расчет себестоимости на основе актуальных цен закупа.
- Учет потерь при копчении/варке (выход готового продукта).

## 4. Авторизация
- Система входа для сотрудников и администратора.
- Ролевая модель (просмотр склада / оформление закупа / доступ к финансовой аналитике).

## 5. Мобильная адаптация
- Интерфейс "Mobile-First".
- Крупные элементы управления для работы "на ходу" (на производстве/складе).
