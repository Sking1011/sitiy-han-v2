# Project: Sytyi Han (Сытый Хан) - ERP для копчения деликатесов

## Суть проекта
Автоматизация учета производства копченой продукции в Казахстане. Основной упор на мобильный интерфейс, точность калькуляции себестоимости и прозрачность прибыли.

## Контекст и цели
- **Валюта:** KZT.
- **Язык:** Русский.
- **Стек:** Next.js, TS, PostgreSQL.
- **Главная цель:** Уйти от таблиц в Excel к удобному веб-приложению.

## Реализованные фичи (для памяти)
- **Партионный учет:** Каждая варка/закуп — это отдельная партия с ID, ценой и остатком.
- **Производство:** Выбор конкретных партий сырья + FIFO списание для специй.
- **Слияние:** Объединение партий внутри категории с расчетом средневзвешенной цены.
- **Списание:** Точечное списание веса с конкретной партии с указанием причины.
- **История:** Двойная запись для слияний (Уход/Приход) для бухгалтерской прозрачности.
